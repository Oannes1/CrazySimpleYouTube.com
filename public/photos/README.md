# Photos folder

Drop photos into the matching subfolder. Filenames are kebab-case, lowercase, descriptive.

## Folder structure

| Folder | What goes here |
|---|---|
| `aaron/` | Headshots, on-camera shots, speaking shots |
| `book/` | Book cover renders, spread shots, lifestyle shots of the book |
| `clients/` | Client headshots (only when release forms signed) |
| `bts/` | Behind-the-scenes filming, editing setup, studio shots |
| `gear/` | Original product photos for the /gear page (mix with Amazon stock) |

## Filename conventions

aaron/aaron-headshot-01.jpg
aaron/aaron-on-camera-vanlife.jpg
book/book-cover-front.jpg
book/book-spread-chapter-17.jpg
bts/filming-setup-vanlife.jpg
gear/sony-a6700-setup.jpg

## Format

- JPG for photos, PNG for screenshots
- Max width 2400px (Next.js will optimize down)
- Avoid HEIC (convert to JPG first)

## Workflow

Drop a photo into the right folder, tell me the filename and what it shows, I will wire it into the matching component.
